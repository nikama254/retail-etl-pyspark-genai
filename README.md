# Retail ETL & Data Mining Project

## Overview

This project demonstrates an end-to-end Data Engineering workflow using PySpark.

## Tech Stack

- Python
- PySpark
- Pandas
- SQLite
- MLxtend
- Databricks

## Business Rules

- Remove InvoiceNo starting with C
- Remove StockCodes: C2, D, M, POST
- Remove records with null CustomerID

## Features

- Data Ingestion
- Data Cleansing
- Data Transformation
- SQLite Loading
- SQL Querying
- Sales Analytics
- Apriori Association Rule Mining

## Dataset

Online Retail Dataset (UCI Machine Learning Repository)

## Results

- Cleaned transaction dataset
- Country-wise sales analysis
- Market Basket Analysis
